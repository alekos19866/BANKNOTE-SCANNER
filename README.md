# BANKNOTE-SCANNER
Complete Data Collection Repository

## Overview
This repository contains ALL data from the BANKNOTE-SCANNER project, consolidated from multiple branches into an organized structure.

## Contents
- **JavaScript Web Application** (`javascript-app/`) - Complete camera-based scanner with UI
- **Python ML Application** (`python-app/`) - Machine learning infrastructure with preprocessing
- **CI/CD Infrastructure** (`.github/workflows/`) - Automated testing
- **Complete Documentation** - Comprehensive data inventory and summaries

## Quick Start
- For web app: Open `javascript-app/index.html` in browser
- For Python app: `cd python-app && pip install -r requirements.txt`
- View `COMPLETE_DATA_SUMMARY.md` for detailed information
- View `DATA_INVENTORY.md` for comprehensive data inventory
